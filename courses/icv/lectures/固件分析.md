# 固件分析

固件是IoT设备的核心构成。分析固件的安全性就像分析IT系统操作系统+应用系统的意义一样。

固件在分析之前，需要从嵌入式设备中提取，其过程有的复杂有点简单。

## 固件分析关注要点

- 口令
- API token
- URL
- 开放服务
- 后门账户
- 配置文件
- 源代码
- 私钥
- 数据存储方式

## 固件分析过程

- 固件提取
- 固件分析
- 文件系统提取
- 文件系统加载
- 文件系统分析
- 基于固件仿真的动态分析
